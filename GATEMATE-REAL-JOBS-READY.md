# GateMate® / Gate Security — Ready-to-Add Real Job Notes

This update deliberately does **not** invent any GateMate® or gate-security jobs for the public Recent Work page. Add genuine work only once you have completed it and have a clear photo or customer permission where needed.

## Send this after a suitable job

- **Town or general area:** e.g. Frinton-on-Sea, St Osyth, Clacton, Tendring village
- **Property type:** home / rental / holiday let / caravan / business
- **What was wrong:** e.g. side gate did not line up, lock had rusted, access was uncontrolled
- **What was fitted or repaired:** describe the actual hardware or work only
- **Security result:** e.g. rear access now locks, keys controlled, gate now closes properly
- **Photo:** one clear before/after or finished image, with no private house numbers if possible
- **Customer review:** only if genuine and approved/visible publicly

## Public Recent Work format

**[Area] — [short outcome]**

A short, factual summary of the issue, the work completed, and the practical benefit. Avoid detailed security-sensitive information such as exact lock configurations, key numbers, full addresses or precise vulnerable access points.
