# Adding Future Job Photos

For every new job photo you want to feature, send the image with a few notes:

- **General area:** e.g. Clacton, Frinton, Harwich, Walton, Tendring village. Do not send a customer address.
- **Service:** e.g. uPVC door alignment, lock change, GateMate-style gate lock, key safe, patio door repair.
- **Problem:** what was wrong before the work.
- **Work completed:** what was repaired, replaced or fitted.
- **Useful outcome:** e.g. door locks smoothly, keys brought under control, rear access more secure.
- **Customer review:** only if a matching genuine review exists and can be quoted.

The website workflow should:

1. Use a real, subtle photo correction only (crop, exposure, sharpness, fast web format).
2. Protect key cuts, key numbers, key-safe codes, addresses, number plates and people where needed.
3. Keep the claim accurate: do not attach a review to a photo unless it is the same job.
4. Add the photo to the relevant service page, Recent Work, the Real Work Gallery and, where enough detail is available, a Case Study.
